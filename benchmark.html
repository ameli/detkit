Benchmark Tests#

A numerical analysis of the performance of the functions are described in Section 4 of [1], which serves as a benchmark test to this package. Here, we provide supplemental details on how to reproduce the numerical results of the paper.

Install Package#

First, install detkit by

pip install detkit

Datasets#

The following numerical results are insensitive to the matrices used during the benchmark tests. However, here we use matrices that are obtained from real applications. The followings describe how to reproduce the datasets described in Appendix C of [1].

Figure

Perform Numerical Tests#

The followings show how to reproduce the results of Section 4 of [1]. First, download the source code of detkit by

git clone https://github.com/ameli/detkit.git

The scripts for the benchmark tests are located at /detkit/benchmark directory of the source code.

1. Run Locally#

  • Run /detkit/benchmark/scripts/benchmark.py to reproduce results for Toeplitz matrices as follows

    cd /detkit/benchmark/scripts
python ./benchmark.py

2. Run on Cluster with Torque#

  • Submit /detkit/benchmark/jobfiles/jobfile_benchmark.pbs to reproduce results of log-determinant of simple matrices:

    cd /imate/benchmark/jobfiles
qsub jobfile_benchmark.pbs

Plot Results#

Run /detkit/benchmark/notebooks/benchmark_plot_draft_3.ipynb to generate plots for computing the log-determinants of Toeplitz matrices. The notebook stores the plots as svg files in /imate/benchmark/plots/ directory.
