detkit#

A toolkit to compute some common functions involving determinant of matrices used in machine learning.

List of Functions#

detkit.logdet

Computes the logdet of a matrix.

detkit.loggdet

Computes the loggdet of a matrix.

detkit.logpdet

Computes the logpdet of a matrix.

See API Reference for a list of all functions.

Features#

  • Functions are implemented with a novel algorithm described in 1.

  • The underlying library is implemented in C++ and wrapped in cython.

  • An accurate count of computational FLOPs during the execution of functions can be measured.

Install#

  • Install with pip:

    pip install detkit

  • Install with conda <https://anaconda.org/s-ameli/detkit>:

    conda install -c s-ameli detkit

  • Pull from docker image

    docker pull sameli/detkit

For a complete install or build instructions, see Install.
